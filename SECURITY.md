# Security Policy

## Supported Versions

We only support the latest version of this project. Please ensure you are running the most recent commit on the `master` branch.

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do **not** report it via a public issue.

Instead, please send an email to the repository owner.

### What to include
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact if exploited.

We will acknowledge receipt of your report within 24 hours and provide a timeline for a fix if applicable.
